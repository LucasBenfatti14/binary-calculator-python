## 🚀 Sobre o projeto

Este projeto consiste em uma calculadora binária desenvolvida em Python, com foco na implementação manual da soma de números binários.  

Diferente de abordagens simples utilizando conversões prontas da linguagem, aqui toda a lógica foi construída do zero, simulando o funcionamento interno de um computador através do uso de carry e processamento bit a bit.

Trata-se de um projeto **100% autoral**, desenvolvido com o objetivo de aprofundar a compreensão sobre lógica de programação e operações de baixo nível.
